pub struct UninitializedUnwindContext<R: Reader>(_);

Common context needed when evaluating the call frame unwinding information.

To avoid re-allocating the context multiple times when evaluating multiple CFI programs, it can be reused. At first, a context is uninitialized (UninitializedUnwindContext). It can be initialized by providing the CommonInformationEntry for the CFI program about to be evaluated and calling UninitializedUnwindContext::initialize. The result is a &mut UnwindContext which borrows the uninitialized context, and can be used to evaluate and run a FrameDescriptionEntry's CFI program.

use gimli::{UninitializedUnwindContext, UnwindTable};

// An uninitialized context.
let mut ctx = UninitializedUnwindContext::new();

// Initialize the context by evaluating the CIE's initial instruction program,
// and generate the unwind table.
let mut table = some_fde.rows(&eh_frame, &bases, &mut ctx)?;
while let Some(row) = table.next_row()? {
    // Do stuff with each row...
}
