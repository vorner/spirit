Register a signal action without checking for forbidden signals.

This acts the same way as register_sigaction, but without checking for the FORBIDDEN signals. All the signals passed are registered and it is up to the caller to make some sense of them.

Note that you really need to know what you're doing if you change eg. the SIGSEGV signal handler. Generally, you don't want to do that. But unlike the other functions here, this function still allows you to do it.
